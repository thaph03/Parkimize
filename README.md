# Parkimize 
Parkimize is a web app that helps users manage and optimize their parking experience. It is bundled with Vite using ReactJS, TailwindCSS, and utilizes Supabase for authentication and database management.

How to run the project:

1. Clone the project using git clone <Project's URL>.
2. Navigate to the main project folder by typing cd.
3. Once in the main project folder, run npm install in the terminal.
4. To start the project using Vite, enter npm run dev in the terminal.
